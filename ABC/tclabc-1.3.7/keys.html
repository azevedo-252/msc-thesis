Key bindings

  • Displacement
    Mouse left-button
    Select a symbol (see also 'selection' below).
    Home
    Go to the beginning of the current voice.
    End
    Go to the end of the current voice.
    Right or MouseWheel down
    Go to the next symbol.
    Left or MouseWheel up
    Go to the previous symbol.
    Shift-right
    Go to the symbol after the next bar.
    Shift-left
    Go to the symbol after the previous bar.
    Down
    Go to the next voice.
    Up
    Go to the previous voice.
    j
    Go to a measure number.
  • Insert symbol
    Q
    Insert a tempo indication.
    n
    Insert a note.
    z
    Insert a rest.
    | (bar) or b
    Insert a bar.
    Return
    Insert an 'extra newline'.
    Z
    Insert a multi-measure rest.
  • Length change
    + (plus)
    Multiply the note length by 2.
    - (minus)
    Divide the note length by 2.
    . (dot)
    Change the note length to a dotted note (toggle).
    : (colon)
    Change the note length to a double dotted note (toggle).
    When the selection is active, if the current symbol is the last one of the selection, 'plus' and 'minus' work on all selected notes and rests.
  • Pitch change
    Prior or PgUp
    Increment the note pitch.
    Next or PgDn
    Decrement the note pitch.
    ' (apostrophe)
    Set the note pitch one octave higher.
    , (comma)
    Set the note pitch one octave lower.
    ^ (circumflex) and _ (underscore)
    Set the note accidental: sharp or flat.
    = (equal)
    Toggle (no accidental) or natural sign.
    When the selection is active, if the current symbol is the last one of the selection, 'Next', 'Prior', 'apostrophe' and 'comma' work on all selected notes and grace notes.
  • Miscellaneous changes
    Space
    Toggle the space indicator (beam breaking).
    c
    Toggle chord edition.
    e
    Edit the symbol details.
    h
    Edit the tune header.
    i
    Edit the MIDI parameters.
    l
    Edit the lyrics (under staff).
    M
    Insert or delete measure bars to have full measures.
    R
    Raw symbol edition.
    Press:
    'Return' to change the symbol (should be the same type).
    'Insert' to insert a new symbol.
    'Escape' to cancel.
    t
    Tie toggle.
    v
    Edit the characteristics of the voices.
    w
    Edit the lyrics (after tune).
    ( (left parenthesis)
    Starting slur toggle.
    ) (right parenthesis)
    Ending slur toggle.
  • Deletion
    Backspace
    Delete the current symbol or selection and go to the previous symbol.
    Delete
    Delete the current symbol or selection.
    When the selection is active, if the current symbol is the last one of the selection, deletion occurs on all symbols of the selection.
  • Recording/playing
    k
    Activate the virtual piano keyboard.
    The next keys work only when MIDI output or input is enabled.
    m
    Voice mute toggle.
    p
    Start or stop playing.
    r
    Start or stop MIDI recording.
  • Program control
    ? or F1
    Display the help window.
    Control-l
    Redisplay.
    a
    Set the 'preview' arguments.
    d
    Toggle displaying the symbol details.
    o
    Open a new file.
    O
    Reload the same file.
    p
    Preview.
    q
    Quit the program.
    s
    Save the tune.
    x
    Export as a MIDI file.
  • Selection and mark
    Mouse left-button (1) and drag.
    Set the selection clipboard.
    Mouse right-button (3).
    Extend/reduce the selection clipboard.
    Mouse middle-button (2).
    Insert the selection clipboard.
    Control-space
    Set the mark at the current symbol.
    Control-w (and Control-x on MS Windows)
    Delete the selection or from mark to current if no selection.
    Control-x Control-x
    Set the mark at the current symbol and go to the previous mark (does not work on MS Windows or with the 'two sel' option).
    Control-y (and Control-v on MS Windows)
    Insert the selection clipboard.
    Escape w (and Control-c on MS Windows)
    Select from mark to the current symbol.
    When the option 'two selections' is set, the system clipboard (PRIMARY on X window) is not accessed by the previous keys. Instead, the following keys must be used:
    Control-c
    Copy the current selection to the system clipboard.
    Control-v
    Insert the system clipboard.
    Control-x
    Copy the current selection to the system clipboard and delete the selection.
